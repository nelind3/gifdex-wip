## Update lexicons:

First install jacquard-lexgen for the `jacquard-codegens` binary:

```bash
cargo install jacquard-lexgen
```

Then run the code generator on the lexicons directory:

```bash
jacquard-codegen --input ../lexicons --output ./src
```
